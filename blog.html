Starting my website....

I started my website by creating a rough layout of what my website should look like using a online software called draw io. This really helped me clearly identify the different sections that my desktop wesbite will have. As you can see through the images I have clearly identied the different sections that my website will have such as the nav, header, main-top, main-bottom and a footer.I have used different colours to clearly differentiate between the different sections.

Development of my website....

Once, I had a rough idea of what my website should like, I started developing my website. I started by creating the index page, which is my homepage, using grid to create the basic layout. Then I created a CSS file and developed the navigation for the website. I was then able to create the several pages of my website. Then I started experimenting with the different colours that I could use on the website, as the overall scheme. After carrying out this experiment several times, I came to conclusion about the colour that I would like to use on my website. To enhance my website I decided to use the colours that were in the same colour family...

Final touches....

Finally, once I had finished adding the content, I decided to experiment with and create keyframe animations. I decided to created a animated background and enhance my website, instead of just using one static colour for the background. Then, experimented with the different gradients, where one gradient, transitioned into another. I used infinite ease to create a smooth transition. The transition will start slowly, then quickly change to a lighter gradient and then slowly end the transition, by transitioning back to its original gradient. I also specied a specific duration, which was 8s. It smoothly transitioned to the specified gradients...
